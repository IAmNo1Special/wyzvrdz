## Using This Sitemap

### When to Use This Reference

1. **Need specific element docs?** -> Look up in UI Elements section, then visit `/documentation/{element_name}`
2. **Need styling help?** -> Check `/documentation/section_styling` or local `styling.md`
3. **Need examples?** -> Visit `/examples` or specific example pages
4. **Having issues?** -> Check `/documentation/section_troubleshooting` or GitHub issues

### URL Patterns

- **Elements**: `https://nicegui.io/documentation/{element_name}`
- **Sections**: `https://nicegui.io/documentation/section_{section_name}`
- **Examples**: `https://nicegui.io/examples#{example_name}`
- **API**: `https://nicegui.io/documentation/{module_name}`

This skill contains detailed information in:

- `SKILL.md` - Core concepts and quick reference
